# Difficulty Service

이 폴더는 **난이도 판별 기능 백엔드 팀원**이 작업합니다.

## 담당 업무
- AI 모델을 활용한 난이도 분석 시스템 개발
- 머신러닝 모델 학습 및 배포
- 난이도 예측 API 구현

## 주요 기능
- 텍스트 난이도 자동 분석
- 문제 난이도 등급 분류
- 학습 수준별 콘텐츠 추천
- 실시간 난이도 예측

## 폴더 구조
- `modules/`: AI 모델 및 핵심 로직
- `api/`: REST API 엔드포인트
- `tests/`: 단위 테스트 및 통합 테스트
- `docs/`: API 문서 및 모델 설명서

## 기술 스택
- FastAPI
- TensorFlow / PyTorch
- Transformers (Hugging Face)
- MLflow (모델 관리)
